# kubernetes-starter

## Vagrant
### 설치
> 설치: `brew install vagrant`  
> 확인: `vagrant --version`  

## VirtualBox
### 설치
> 설치: [Download VirtualBox](https://www.virtualbox.org/wiki/Downloads)  
> Apple M1(ARM) 의 경우 brew 에서 VirtualBox 를 설치할 수 없고 다운로드 페이지를 통해서 Developer preview 를 다운받는다.(2022-11-18 기준)
